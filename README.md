# Know Your Plastic Footprint

Know Your Plastic Footprint is an interactive R Shiny application designed to help individuals quantify their plastic consumption and reflect on their environmental impact.

Users input their weekly or monthly use of common plastic items (e.g., bottles, sachets, packaging), and the app computes:
- Weekly plastic footprint
- Monthly plastic footprint
- Annual plastic footprint
- Potential reduction based on user-defined goals

The application supports sustainability campaign (i.e. Way Plastikay) by transforming awareness into measurable action.

## Features

### Plastic Footprint Calculator
- Estimates plastic use based on common items
- Converts usage into grams/week, month, and year

### Reduction Goal Setting
- Users define their target reduction (%)
- Calculates potential plastic avoided annually

### Interactive Dashboard
- Aggregated results visualization
- Distribution of plastic use by type
- Reduction goal trends

### Local Data Storage
- Stores responses in a CSV file
- No external database required
- Works offline or in restricted environments

### Admin Panel
- Download collected data
- Monitor submissions
